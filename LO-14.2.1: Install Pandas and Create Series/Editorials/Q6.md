## 1. Problem Description
Implement a function to convert a list into a labeled Series dynamically.

## 2. Objective
Apply Series creation logic with dynamic indexing.

## 3. Hint
You can use a list comprehension or generator to create the 'DayX' labels.

## 4. Short Explanation
Loop through the range of the list length to create keys like `f'Day{i+1}'`.

## 5. Detailed Explanation
```python
import pandas as pd

def analyze_sales(sales_list):
    # Generate index labels dynamically based on input length
    labels = [f"Day{i+1}" for i in range(len(sales_list))]
    
    # Create the Series
    s = pd.Series(sales_list, index=labels)
    
    return s

# Test
print(analyze_sales([100, 200, 150]))
```
