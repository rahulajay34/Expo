## 1. Problem Description
Identify the correct command to install the pandas library using the pip package manager.

## 2. Objective
To test knowledge of package installation syntax.

## 3. Hint
The package name on PyPI is usually the same as the import name.

## 4. Short Explanation
The correct command is `pip install pandas`.

## 5. Detailed Explanation
`pip` is the standard package installer for Python. The syntax is `pip install <package_name>`. For Pandas, the package name is simply 'pandas'.

Answer: B
