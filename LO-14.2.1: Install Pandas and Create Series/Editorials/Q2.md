## 1. Problem Description
Create a Series with custom index labels.

## 2. Objective
Demonstrate ability to create a basic Series with explicit index.

## 3. Hint
Use the `index` parameter in the `pd.Series()` constructor.

## 4. Short Explanation
Pass the data as the first argument and the labels list to the `index` argument.

## 5. Detailed Explanation
```python
import pandas as pd

temps = [98.6, 99.1, 100.2, 97.9]
days = ['Mon', 'Tue', 'Wed', 'Thu']

s = pd.Series(temps, index=days)
print(s)
```
Output:
```text
Mon     98.6
Tue     99.1
Wed    100.2
Thu     97.9
dtype: float64
```
